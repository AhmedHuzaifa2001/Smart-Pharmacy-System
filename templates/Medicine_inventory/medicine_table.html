Bulk Upload Medicines

Upload multiple medicines at once with images support

{% csrf_token %}

📋 Bulk Upload with Images Instructions:

Step 1: Prepare Your Images
  • • Place all medicine images in: /media/medical_products/
  • • Supported formats: JPG, PNG, WEBP
  • • Recommended size: 500x500px or larger
  • • Use clear, descriptive filenames (e.g., paracetamol.jpg)
Step 2: Update Your CSV
  • • Download the template below (includes image_path column)
  • • Fill in medicine data as usual
  • • In image_path column, use just the filename: paracetamol.jpg
  • • Leave image_path empty if no image available
Step 3: Upload CSV
  • • Upload your completed CSV file below
  • • System will automatically link images during processing
  • • Medicines without valid image paths will be created without images

CSV Example:

name,category,...,image_path
Paracetamol 500mg,Analgesic,...,paracetamol.jpg
Amoxicillin 250mg,Antibiotic,...,amoxicillin.png
Vitamin C 1000mg,Vitamin,...,(empty - no image)

CSV files only, up to 5MB

How it works:

With Images: CSV includes image_path column pointing to files in /media/medical_products/

Without Images: Leave image_path empty, medicines will show default "no image" placeholder

Mixed: Some medicines with images, some without - all perfectly supported!
